simple framework draft
(MVC - Plugin oriented)

__dependencies__: backbone, requirejs, underscore, jquery (other deps should be added through plugins)

## install

```sh

$ cd path/to/project
$ npm install
$ bower install
$ node app.js

```

the main file is `public/js/main.js`
